# alimentacion-balanceada-para-todos
proyecto de analisis sobre el costo y asequibilidad de una dieta saludable usando datos de FAOSTAT
Objetivo del Proyecto 
Evaluar si una dieta saludable es accesible para la poblacion  colombiana utilizando datos oficiales de FAOSTAT, analizando el costo, la asequibilidad y los indicadores relacionados con la capacidad economica de la poblacion.
Preguntas de investigacion
Cuantas personas en Colombia no pueden pagar una dieta saludable?
Que porcentaje de la poblacion representa?
Cuanto cuesta una dieta saludable por persona por dia?
Como se compara este costo con otros paises de la region?
Que implicaciones tiene esto para la alimentacion balanceada?
